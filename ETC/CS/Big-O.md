# Big-O 표기법
> 시간 복잡도를 표현하는 표기법

### big-O, big-Ω, big-θ 이게 뭔데?
- big-O: 시간의 상한을 나타낸다.
- big-Ω: 시간의 하한을 나타낸다.
- big-θ: 딱 맞는 시간을 나타낸다.

1. O(b)
2. O(b)
3. O(1)
4. O(a/b)
5. O(log n) 절반씩 줄어든다 - 1/2씩 줄어든다.
6. O(루트 n)
7. O(array.length ^2) 결국에는 이중 for 문. 배열 개수 카피 저장
8. O(log n)
9. O(n^2) : 반복 한 번 재귀해서 반복 두 번
10. O(n log2 n)
